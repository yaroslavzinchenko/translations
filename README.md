# Музыкальные тексты

### Ссылки:

https://yaroslav-zinchenko.000webhostapp.com/translations - старая версия. Скоро появится новая.

### Описание
Веб-приложение. Позволяет просматривать, добавлять, удалять и изменять музыкальные тексты.

### Используемые технологии:

* HTML
* SCSS
* PHP
* Laravel
* MySQL
* JavaScript
* jQuery
* Github
* Github Pages
